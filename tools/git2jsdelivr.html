GitHub to jsDelivr is a utility that links static assets (static assets) from GitHub to jsDelivr's free CDN. You can link most of the git on GitHub to jsDelivr to take advantage of the powerful CDN from this service.

About jsDelivr

jsDelivr is a free, fast and stable open source CDN service. jsDelivr was born to support CDN for open source software: npm and GitHub. Most of the git on GitHub comes with CDN support from js Delivr.

Outstanding Features

Some outstanding features of jsDeliver:

  • jsDelivr is supported by 4 big CDNs: Cloudflare, Fastly, Bunny, Quantil.
  • There are more than 750 global hotspots.
  • Supports automatic load balancing.
  • Faster than most other CDNs.
  • High reliability, stable operation.
  • Works with both in China.

Valid GitHub link

Valid GitHub links look like the example below:

  • Github file URL: https://github.com/twbs/bootstrap/blob/v4-dev/dist/css/bootstrap.min.css
  • Github file URL: https://github.com/twbs/bootstrap/raw/v4-dev/dist/css/bootstrap.min.css
  • Github file URL: https://github.com/twbs/bootstrap/blame/v4-dev/dist/css/bootstrap.min.css
  • Github raw file URL: https://raw.githubusercontent.com/twbs/bootstrap/v4-dev/dist/css/bootstrap.min.css
